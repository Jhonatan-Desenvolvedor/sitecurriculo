Front-End

HTML, ou Linguagem de Marcação de Hipertexto, é a base para criar páginas da web, definindo a estrutura e o conteúdo de documentos online. Ele utiliza tags (marcadores) para organizar elementos como títulos, parágrafos, imagens e links, e é a linguagem de estrutura, enquanto CSS é para o estilo (visual) e JavaScript é para a interatividade.

CSS é uma linguagem de estilo que separa o conteúdo de um site (HTML) de sua apresentação visual. Ele é usado para definir a aparência de documentos web, permitindo personalizar cores, fontes,layouts, e outros elementos de design, tornando o site visualmente agradável e consistente.

JavaScript é uma linguagem de programação versátil usada para tornar páginas web interativas, adicionando funcionalidades como menus dinâmicos, animações e formulários. É interpretada, leve e baseada em objetos, executada tanto no navegador do cliente quanto em servidores, sendo uma das tecnologias centrais da World Wide Web.

Angular é um framework de código aberto desenvolvido pelo Google para a construção de aplicativos web dinâmicos e interativos. Ele se baseia em uma arquitetura orientada a componentes, usa TypeScript e é amplamente utilizado para criar Single-Page Applications (SPAs).

Back-End

Java é uma linguagem de programação orientada a objetos, multiplataforma e robusta, utilizada no desenvolvimento de uma vasta gama de aplicações, desde software empresarial e sistemas operacionais para smartphones até inteligência artificial.

C# é uma linguagem de programação de alto nível, moderna, multiparadigma e orientada a objetos, desenvolvida pela Microsoft como parte da plataforma .NET. É usada para criar uma ampla gama de aplicações, como desenvolvimento web, jogos, aplicativos móveis e serviços de nuvem.
